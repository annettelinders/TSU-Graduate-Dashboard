
HOW TO RUN THE TSU GRADUATE OUTCOMES DASHBOARD
==============================================

This dashboard allows you to interactively explore graduate outcomes by major and college.

REQUIREMENTS:
-------------
- Python 3.8+
- Install packages:
    pip install streamlit plotly pandas

FILES INCLUDED:
---------------
- TSU_Graduate_Summary.csv : cleaned data file
- tsu_dashboard_prototype.py : Streamlit app file

HOW TO RUN:
-----------
1. Open terminal or command prompt.
2. Navigate to folder containing these files.
3. Run the following command:
    streamlit run tsu_dashboard_prototype.py

This will open the dashboard in your default web browser.
