
TSU Graduate Outcomes Dashboard
===============================

This Streamlit dashboard allows you to explore graduate outcome metrics by degree program.

FILES INCLUDED:
---------------
- TSU_Graduate_Summary_Updated.csv
- tsu_dashboard_updated.py

HOW TO RUN:
-----------
1. Install required packages:
    pip install streamlit pandas plotly

2. In terminal, navigate to the folder with the files:
    cd your-folder

3. Run the app:
    streamlit run tsu_dashboard_updated.py

You'll see the dashboard open in your browser!
